# BMI Calculator
## Project Overview
The BMI (Body Mass Index) Calculator is a Python-based application that calculates a user's BMI using their weight and height. It classifies the result into standard health categories such as underweight, normal, overweight, and obese. The project is available in both command-line and GUI versions, making it suitable for beginners and advanced users.

## Key Features
* BMI calculation using standard formula
* Health category classification
* Input validation and error handling
* Graphical User Interface (GUI) using Tkinter
* Data storage using JSON
* BMI history tracking
* Graphical visualization using Matplotlib

## Technologies Used
* Python
* Tkinter
* Matplotlib
* JSON

## Installation
1. Clone the repository:
   git clone https://github.com/your-username/bmi-calculator.git
2. Navigate to the project folder:
   cd bmi-calculator
3. Install required dependencies:
   pip install matplotlib

## Usage
Run the command-line version:
python bmi_cli.py
Run the GUI version:
python bmi_gui.py

## BMI Formula
BMI = weight / (height * height)
