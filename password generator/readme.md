# Random Password Generator
## Project Description
The Random Password Generator is a Python-based application that generates secure and customizable passwords based on user-defined inputs. The application allows users to specify password length and select character types such as letters, numbers, and symbols. It is implemented in both command-line and graphical user interface (GUI) versions to demonstrate different approaches to user interaction and software design.

## Key Features
* Generate random and secure passwords
* User-defined password length
* Support for letters, digits, and special characters
* Input validation and error handling
* GUI-based password generator using Tkinter
* Clipboard integration for copying generated passwords
* Implementation of basic password complexity rules

## Technologies Used
* Python 3
* Tkinter (GUI development)
* Random module
* String module

Run the command-line version:

Run the GUI version:
